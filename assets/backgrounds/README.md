# خلفيات التطبيق

ضع صورك هنا بهذه الأسماء بالضبط:

| الملف | الوصف |
|-------|--------|
| `dark.jpg` | خلفية داكنة |
| `midnight.jpg` | خلفية ليلية |
| `ocean.jpg` | خلفية محيطية |
| `forest.jpg` | خلفية غابية |
| `royal.jpg` | خلفية ملكية |
| `sunset.jpg` | خلفية غروب الشمس |
| `space.jpg` | خلفية فضائية |
| `charcoal.jpg` | خلفية فحمية |

## مواصفات الصور:
- **الصيغة:** JPG أو PNG
- **الأبعاد:** 1080x1920 (9:16) أو 720x1280
- **الحجم:** أقل من 500KB لكل صورة

إذا ما عندك صور، التطبيق راح يعطي خطأ. ممكن تحمل صور مجانية من:
- Unsplash.com
- Pexels.com
